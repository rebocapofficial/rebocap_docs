Rebocap 官方技术名词多语言对齐备忘表 (Rebocap Term Glossary)

一、 核心硬件设备名词 (Hardware Components)

【繁体中文】 追蹤器

【英    文】 Sensor
【日    文】 センサー
【繁体中文】 感測器
【说    明】 物理学与传感器技术相关表述

【简体中文】 接收器 / USB 接收器
【英    文】 Receiver / USB Receiver
【日    文】 レシーバー / USBレシーバー
【繁体中文】 接收器 / USB 接收器
【说    明】 用于插在电脑上的 USB 无线狗

【简体中文】 充电盒
【英    文】  Charging Case  //注释：英语统一为Charging Case
【日    文】 充電ボックス
【繁体中文】 充電盒
【说    明】 15点套装自带的收纳充电盒

三、 校准与软件设置名词 (Calibration & Settings)
----------------------------------------------------------------------------------------
【简体中文】 陀螺仪校准
【英    文】 Gyroscope Calibration
【日    文】 ジャイロスコープ校正
【繁体中文】 陀螺儀校準
【说    明】 追踪器静置在地面上的零偏校准

【简体中文】 磁场校准
【英    文】 Magnetic Field Calibration
【日    文】 磁場校正
【繁体中文】 磁場校準
【说    明】 追踪器装入托盘中进行空间磁场分布记录

【简体中文】 简易磁场校准
【英    文】 Simple Magnetic Calibration
【日    文】 簡易磁場校正
【繁体中文】 簡易磁場校準
【说    明】 通过拿在手中手腕划8字完成的便捷磁校准

【简体中文】 磁场读数
【英    文】 Magnetic Readings / Magnetic Field Readings
【日    文】 磁場読数
【繁体中文】 磁場讀數
【说    明】 软件中显示的每个追踪器当前受磁场干扰的值

【简体中文】 校准姿态图鉴 / VR校准姿态图鉴
【英    文】 Calibration Posture Guide / VR Calibration Posture Guide
【日    文】 校正ポーズ図鑑 / VR校正ポーズ図鑑
【繁体中文】 校準姿態圖鑑 / VR校準姿態圖鑑
【说    明】 A Pose、T Pose、S Pose、B Pose 等身体姿势参考

【简体中文】 AI引擎
【英    文】 AI Engine
【日    文】 AIエンジン
【繁体中文】 AI引擎
【说    明】 针对未穿戴足部追踪器时自动预测姿态的防滑步悬空算法

【简体中文】 配置 SteamVR 输出节点
【英    文】 Set SteamVR Nodes
【日    文】 SteamVR 出力ノード設定
【繁体中文】 配置 SteamVR 输出节点
【说    明】 用来设置哪些追踪点在steamvr中显示或隐藏
