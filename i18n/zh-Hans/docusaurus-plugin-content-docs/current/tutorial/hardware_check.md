## 开箱检查

- 产品目前采用抽屉彩盒+普通纸箱包装，中间采用气柱垫6面环绕来增加缓冲
- 产品清单

  - 跟踪器 x 15

    > 每个跟踪器底部有标签，标签上绘制有对应的数字编码以及身体部位，比如1号就是左上腿，3号是左下腿。跟踪器的位置都是固定的，除非你使用[替换功能](../ui_help_doc/remap#trackerreplace)
  - 接收器 x 1
  - 充电底仓 x 1

    > 可用于收纳，本身不包含电池

:::danger 使用12V以上电源适配器（如24V电源适配器），充电主板会因为过压导致烧坏！！！

:::

- 电源适配器 x 1

  > 12V 2A，宽电压，兼容 110V 和 220V

- 绑带 x 15 + 收纳网袋 x 1

  1. 100cm \* 2
  2. 60cm \* 3
  3. 40cm \* 6
  4. 25cm \* 4

- 抽屉彩盒 x 1

- 【赠品】15个身体部位贴纸 x 1

  > 辅助查看每个跟踪器的位置，方便佩戴以后二次检查是否佩戴出错。

- 【赠品】塑料卷尺 x 1

- 【加价选配】快拆 x 15

  > 在中国大陆地区属于可选配件（即中国淘宝官方店铺购买），对于非中国大陆地区，由于配送距离较远，为防止部分用户因为不清楚快拆作用或者忘记购买导致二次购买产生较多配送费，因此会作为必选配件和主产品一起售卖。

- 产品开箱视频

  > 如何打开充电盒上盖，请具体查看视频第13秒，大拇指顶住盒子弹片按钮最上端，另一只手开启上盖。

- 关于硬件品质说明

  1. 跟踪器上下盖子之前存在一定间隙，以及上下盖子存在小幅度断差，均属于正常现象
  2. 大的充电盒子中，中托部分注塑出水口留在外观面，导致卡扣旁边有一定缝隙和瑕疵。
  3. 接收器的USB接口部分出厂歪了一点点，不影响任何使用。
  4. 塑料外壳目前均没有进行UV电镀，容易留下划痕
  5. 不同跟踪器电池存在一定容量误差属于正常现象，以使用时长为准【电量百分比只能作为参考，不可作为评估指标】。

## 充电和使用时长说明

- 首次拿到需要确认是否可以充电，正常情况下，适配器插上以后所有跟踪器亮红灯，充满电后转变为绿灯
- 如果发现没有亮灯，请检查是否是跟踪器运输途中松动，重新插拔相应的跟踪器，如果全部没有亮灯，检查电源适配器是否通电
- 正常情况下，充满电约需2小时，使用时长约为30小时


:::info 使用时长说明和充电说明
1. 使用时长和环境温度有关，低温会降低使用时长
2. 如果个别节点没有正常关机，电量持续消耗，导致和其它跟踪器使用时长相差较大
3. 充满电后，拔出跟踪器再次插入，会再次显示为红灯状态属于正常现象，类似传统手机或者其它电器充电，满电后拔出再次充电依然可以进行短时间涓流充电。
:::

- <p>极个别跟踪器可能无法在充满电后转换为绿灯状态，请以连接后软件显示的电量为准！</p>
- <p>长时间放置时，建议每隔2~~3个月放电后充电一次，否则电池寿命将会收到影响！例如，VR用户可能会有5~~7个闲置点位，未来若有损坏的点位可使用小臂替换，因此建议做好电池保养工作。</p>

> 注：使用时长与人体运动状态无关，只要连接后，无论动作快慢，消耗的电流大小保持一致。

## 跟踪器状态说明

|       |   待机状态  |   断开状态   | 连接状态 |       电量低       | 固件更新 | 固件更新成功 |
| :---: | :-----: | :------: | :--: | :-------------: | :--: | :----: |
| RGB状态 | 呼吸灯(3S) | 呼吸灯(10S) | 快速闪烁 | 红灯闪烁<br />3次后关机 | 蓝灯闪烁 |  白灯闪烁  |

- 跟踪器点亮以后，处于待机状态，待机状态下呼吸灯持续闪烁30S，10分钟未连接则自动关机
- 接收器只要通电，跟踪器就会进入连接状态，连接状态下不会自动关机

:::warning 关闭软件不会自动关机
部分情况下关机失败也会导致持续耗电！
:::



- 如果接收器断电，则跟踪器会再次转换到待机状态
- 跟踪器未自动关机情况下，接收器通电后，跟踪器将会立即进入连接状态
