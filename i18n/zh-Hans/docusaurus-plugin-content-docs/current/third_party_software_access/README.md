### [云镜虚拟直播](https://fancyjing.com/home)

- 简介
> 「云镜虚拟直播」是邻境科技旗下的一款免费3D虚拟直播工具。只需一台手机或是带有摄像头的电脑，即可实现面部、半身及全身动捕，同时已经接入 `Rebocap` 动捕设备实现精确动作捕捉。内置丰富道具、舞蹈动作及玩法，为虚拟主播提供更便捷的直播服务。
> 
> 注意：如果出现模型胸部和腰部拉长现象，请联系建模师修改骨骼，一般是由于`Q`版模型胸部骨骼没有和`mesh`保持一致缩放导致

- 视频教程
> [点击这里跳转查看](https://www.bilibili.com/video/BV1z7hFzfEw6)

- 接入文档
- 加 `QQ` 群联系官方客服，群号见[网页底部](https://fancyjing.com/home)
