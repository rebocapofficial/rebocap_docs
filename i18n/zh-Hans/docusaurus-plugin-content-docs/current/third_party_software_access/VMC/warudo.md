# warudo 接入
配置 VMC 接收即可，其它设置不要动，如果动了，请重新设置，先查看只有 rebocap 接入的效果，另外，一定要上传骨架到 rebocap 中，[见这里](README#vmc_instroction)。否则容易导致脚底悬空，或者整体运动跟随问题较大。

下边是截图流程图，非常简单，VMC 接收，除了图中标注的 1、2 外，如果你不清楚你修改的配置是什么，那么不要修改！

其中第三张图的标注信息需要自己修改：

1. 选择具体VMC接收以后，映射的角色
2. VMC端口号，必须和 rebocap 中的 vmc 输出端口号保持一致，[具体见这里的配置](../../ui_help_doc/control/connect#cal_pc_panel)

- 一定要打开 VMC输出 开关【在 PC 面板中】
- 如果本页面图片无法放大，刷新以后点击即可放大
- 如果配置相同还是无法完成动捕，请移除 warudo 其它所有配置，比如面部捕捉、手部捕捉，然后再按照以上三张图配置接入。否则会因为冲突导致无法动捕，冲突需要自行解决或者自行咨询 warudo 官方。
