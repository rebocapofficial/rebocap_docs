# 传感器整体预览

如下图所示，对应关节部位绿色则代表传感器已连接，灰色则表示未连接。图中左侧即为人体自身的左侧，比如下图中左脚的传感器没有连接，右边下腿的传感器也没有连接。

## 传感器替换功能
这里可以通过传感器点位拖拽，实现传感器重组

:::warning 替换以后，如果没有点亮，会导致校准失败


很多用户不小心拖拽以后，比如将左手拖拽到左上腿，但是校准的时候又没有开启左手，会提示校准失败，因为左上腿没有开启！！！
拖拽以后，被替换的点位不管有没有被点亮，都是不生效的。

:::


### 拖拽演示
> 在节点上鼠标左键并保持，拖拽到目标点位即可。

### 能够替换其它部位的点位
   * 左手   (13号节点)
   * 右手   (14号节点)
   * 左下臂 (11号节点)
   * 右下臂 (12号节点)


### 能够被替换的部位
   全部


### 特别功能
   * 替换为肩膀位置
    > 这里如果希望肩膀部位更加灵活，可以替换为肩膀位置，但是对绑带要求比较高，需要自行解决。


### 替换示例
如下图所示，左右手四个点位全部替换了，替换如下部位
* 左肩
* 右肩
* 左下腿
  > 左下腿替换以后没有被点亮，因此会发生校准失败的现象！！！
* 右下腿
