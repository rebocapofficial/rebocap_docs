# 硬件列表信息
硬件列表信息用于全局预览。
- 第一列
  > 部位名称
- 第二列
  > 部位编号，和传感器后面的标牌中的编号对应
- 第三列
  > 电池电量，电量百分比只能用于参考，这里是通过电压转换而来，但是电压测量不是专业 A/D 采样测量，因此有一定误差，不是绝对百分比，以具体使用时长为准
- 第四列
  > 这里指的是通信质量，比如最近2秒中，120个帧率数据有多少百分比接收成功，**不是信号强度**！信号强度在跟踪器详细信息中查看，单位是 `dbm`，一般 >-70dbm 认为信号良好，比如-30 dbm 就是很强的信号
- 第五列
  > 磁场绝对值大小，单位是 uT，磁场一般一致性好就是显示绿色，如果个别很高或者很低会显示红色，磁场怎么看，具体请查阅 磁场检测和诊断

# 硬件详细信息
点击列表中某一项，即可查看硬件详细信息。

<a id="close_single_tracker"></a>

### 单独关机
  > 可以点击关机按钮，单独关闭这个跟踪器

<a id="detail_information"></a>

### 其它信息
- 电池电量
  > 具体电量百分比，这里仅仅用于大致参考，具体以使用时长为准
- 通信质量
  > 最近2秒中，120个帧率数据有多少百分比接收成功，不是信号强度！信号强度在跟踪器详细信息中查看，单位是 `dbm`，一般 >-70dbm 认为信号良好，比如-30 dbm 就是很强的信号
- 接收信号
  > 接收器收到的传感器的信号强度，>-70 dbm 认为是信号良好
- 被动信号
  > 传感器收到的接收器的信号强度，>-70 dbm 认为是信号良好
- 相对磁场
  > 这里是相对于校准时 A-Pose 所在空间和时刻的相对大小，相对大小持续在 1.1 以内一般认为磁场环境良好，磁场环境良好的情况下，应用抗磁算法效果更好，具体磁场判定和磁场校准，请看这里
- 加速度
  > 三轴校准后加速度，归一化到当前重力加速度中
- 陀螺仪
  > 三轴校准后陀螺仪，如果跟踪器完全静止，这里的数值离0点超过0.2，建议进行 陀螺仪校准
