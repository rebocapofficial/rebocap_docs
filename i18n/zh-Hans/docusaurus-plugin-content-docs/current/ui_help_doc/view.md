# 3D 预览功能
- **展示动捕实时预览画面**

  **操作说明**
  * 鼠标左键移动视角
  * 鼠标中键拖动相机位置，也就是拖动视角
  * 鼠标滚轮是按照当前方向，向前向后移动相机位置，可以用于调节视图中人物大小
- **展示跟踪器**
    > 下图 `3d_preview` 中标注的 1 区域是用于展示跟踪器，当跟踪器被选中以后，这里可以实时展示跟踪器方向，点击盒子按钮以后，则关闭跟踪器实时预览

<a id="how_to_close_3d_preview"></a>

### 如何关闭3D节省性能
* 点击下图 `3d_preview` 中2区域按钮，可以暂停渲染，暂停渲染能够节省资源
* 彻底关闭3D 渲染方法：配置页面中可以直接关闭3D渲染，重启后生效，能够节省内存，更大程度节省 CPU 和 显卡 资源
