## 必須理解項目
- [磁場校正](../ui_help_doc/control/config#magnet_calibrate)

:::danger 初回充電時には通常磁場校正が必要で、後続は一定期間ごとに磁場校正を行うことをお勧めします


- 磁場校正については、文書内（上記の磁場校正リンクの下）とコミュニティと協力して作成した校正ビデオを参考にしてください。
- 磁場校正は必ず習得し、その注意事項は非常に重要です。注意事項に従わない場合、校正結果が誤る可能性があり、磁場校正は何度でも行うことができます。

:::

## ビデオチュートリアル
動画チュートリアルは補助的なオプションであり、ドキュメントの説明の方がより詳細で、工程中に発生する問題についても取り上げています。また、動画チュートリアルは基礎的な内容であり、[SteamVR 及びその他のサードパーティ配信ソフトとの接続はこちらを参照してください](https://doc.rebocap.com/ja_JP/tutorial/connect_and_use.html#third_party)、[Blender、Unity、UE などのプラグインはこちらを参照してください](../plugins/plugins) のようなサードパーティソフトへのリンクは含まれていません。さらに疑問がある場合は[コミュニティで質問](../README#community)してください。

:::danger より良い効果を得たい場合や様々な問題に直面した場合は、文字版のチュートリアルを完全に読むことをお勧めします


- 例えば、足元の装着方法が大きく影響します
- 例えば、磁気耐性をオンにする必要があるかどうか
- 例えば、AIエンジンをオンにする必要があるかどうか
- 他のソフトウェアとどう接続するか

:::
