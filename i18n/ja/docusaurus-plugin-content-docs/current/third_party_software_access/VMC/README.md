# VMC 使用
vmc プロトコルは非常に簡単に使用できます。キャリブレーション後に vmc プロトコルを開くだけで、[具体的にはこちらを参照してください](../../ui_help_doc/control/connect#cal_pc_panel)、その後、他のソフトウェアで受信を設定します。VMCは汎用のモーションキャプチャプロトコルです。[詳細はこちらをご覧ください](https://protocol.vmc.info/english.html)

もしあなたが配信者で、使用しているソフトウェアが `rebocap` をサポートしているか不明な場合は、あなたのソフトウェアが vmc プロトコルをサポートしているか確認してください。もちろん、あなたのソフトウェアがサポートしていない場合は、開発者に連絡して私たちが提供する [SDK](../../SDK/README) を使用して接続するか、直接私たちが提供する[プラグイン](../../plugins/plugins)を使用して接続してください。

:::info VMC プロトコルユーザーへの注意事項


VMCプロトコルユーザーは、スケルトンをアップロードすることを強く推奨します。VRMモデルの場合は、スケルトンを直接 rebocap にアップロードできます。[こちらがスケルトンアップロードの紹介です](../../ui_help_doc/control/skeleton_setting#skeleton_import)。他のフォーマットのユーザーは、[blender プラグインを使用してスケルトンをエクスポート](../../plugins/blender#skeleton_export)できます。エクスポートされたファイルは json ファイルで、手動で修正できます。

もしこの2つの方法がわからない場合は、あなたのバーチャルキャラクターのスケルトンの高さを知る必要があります。VMC のスケール設定を調整し、rebocap の身長 * vmc_scale = 現在のバーチャルキャラクターの身長に設定しますが、この効果は一般的にあまり理想的ではありません。

:::
