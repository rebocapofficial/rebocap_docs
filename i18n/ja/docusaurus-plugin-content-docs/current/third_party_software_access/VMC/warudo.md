# warudo 接続
VMCを受信するように設定するだけで、他の設定は変更しないでください。変更した場合は、再設定が必要です。まずはrebocapのみの接続効果を確認してください。また、必ず骨格をrebocapにアップロードしてください。[こちらを参照](README#vmc_instroction)。そうしないと、足元が浮いたり、全体の動きに大きな問題が生じる可能性があります。

下記はスクリーンショットのフローチャートで、非常に簡単です。VMCを受信する際、図中の1、2以外の設定について不明な場合は、変更しないでください！

3枚目の図の注釈情報は自分で修正する必要があります：

1. 具体的なVMC受信後、マッピングするキャラクター
2. VMCポート番号は、必ずrebocapのvmc出力ポート番号と一致させてください。[具体的な設定はここを参照](../../ui_help_doc/control/connect#cal_pc_panel)

- 必ずVMC出力スイッチをオンにしてください【PCパネル内】  
- このページの画像が拡大できない場合は、リフレッシュ後にクリックしてください  
- 設定が同じでも動捕が完了しない場合は、warudoの他のすべての設定を削除してください。例えば、顔のキャプチャや手のキャプチャなど、その後、上記の3枚の図に従って接続を設定してください。そうしないと、競合が原因で動捕ができなくなります。競合は自分で解決するか、warudoの公式に相談してください。
