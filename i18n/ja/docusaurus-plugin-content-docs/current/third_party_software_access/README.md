# その他の接続

### [雲鏡バーチャルライブ配信](https://fancyjing.com/home)

- 概要
> 「雲鏡バーチャルライブ配信」は、隣境科技が提供する無料の3Dバーチャルライブツールです。スマートフォン1台、またはカメラ付きのPCだけで、顔・上半身・全身のモーションキャプチャを実現でき、さらに `Rebocap` モーションキャプチャデバイスと連携して精密な動作捕捉を行えます。豊富な小道具・ダンスモーション・ゲームプレイを内蔵し、VTuber により便利な配信サービスを提供します。
> 
> 注意：モデルの胸部や腰部が伸びてしまう場合は、モデラーに依頼してボーンを修正してください。通常は、`Q`版モデルの胸部ボーンのスケールが `mesh` と一致していないことが原因です。

- ビデオチュートリアル
> [こちらをクリックしてご覧ください](https://www.bilibili.com/video/BV1z7hFzfEw6)

- 導入ドキュメント
- `QQ` グループに参加して公式カスタマーサービスに連絡してください。グループ番号は[ウェブページ最下部](https://fancyjing.com/home)をご覧ください


### [浩嘉亿和バーチャルヒューマン配信](https://sk157zmgy6.feishu.cn/docx/RK4yduZF1oEhuQxKI8scu4h2nYg)
> 浩嘉亿和ライブギルドは、エンタメ・音声・二次元・バーチャル分野の配信者を対象とした専門ギルドです。配信者トレーニング、配信シーン構築、バーチャルヒューマン制作、配信者専用の運営プランニングなど、オンラインとオフラインを組み合わせた多段階の専門的な運営サービスを提供します。プロフェッショナルチームがバーチャル新勢力をさらに強化し、C エンドと B エンドのクライアントに高品質なフルスタックのバーチャルヒューマン制作サービスを提供します。経験豊富なテクニカルアーティスト（Technical Artist）がリーダーとなり、IP 企画・デザイン・運営・制作・プログラムを一体化した技術サポートチームを構成しています。

- 導入ドキュメント
