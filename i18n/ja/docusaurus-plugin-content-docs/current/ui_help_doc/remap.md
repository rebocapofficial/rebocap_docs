# センサー全体プレビュー

下図のように、対応する関節部位が緑色の場合はセンサーが接続されていることを示し、灰色の場合は接続されていないことを示します。図の左側は人体自身の左側を示しています。例えば、下図では左足のセンサーが接続されておらず、右側の下腿のセンサーも接続されていません。

![img.png](/img/2d_view.png)

## センサー交換機能
ここではセンサーの位置をドラッグすることで、センサーの再配置が可能です。

> **[警告] 交換後、点灯しない場合、キャリブレーションに失敗します**
> 
> 多くのユーザーが誤ってドラッグしてしまい、例えば左手を左上腿にドラッグした場合、キャリブレーション時に左手をオンにしないと、キャリブレーションに失敗したと表示されます。なぜなら左上腿がオンになっていないからです！！！
> ドラッグ後、交換された位置は点灯していない限り、効果がありません。

### ドラッグ＆ドロップ デモ
> ノード上でマウスの左ボタンを押したまま、目的の位置までドラッグしてください。

### 他の部位に交換可能な位置
   * 左手 (13番ノード)
   * 右手 (14番ノード)
   * 左前腕 (11番ノード)
   * 右前腕 (12番ノード)


### 交換可能な部位
   全部


### 特別機能
   * 肩の位置に交換
    > ここで肩の部位をより柔軟にしたい場合、肩の位置に交換できますが、バンドの要求が高く、自分で解決する必要があります。

### 交換例
下図のように、左右の手の4つの位置がすべて交換され、以下の部位に交換されました。
* 左肩
* 右肩
* 左下腿
  > 左下腿は交換後に点灯していないため、キャリブレーションに失敗する現象が発生します！！！
* 右下腿
