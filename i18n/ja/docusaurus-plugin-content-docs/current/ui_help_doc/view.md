# 3Dプレビュー機能
- **モーションキャプチャのリアルタイムプレビュー画面を表示**

  **操作説明**
  * マウスの左ボタンで視点を移動
  * マウスの中ボタンでカメラ位置をドラッグ、つまり視点をドラッグ
  * マウスホイールで現在の方向に沿ってカメラ位置を前後に移動し、ビュー内の人物の大きさを調整可能
- **トラッカーを表示**
    > 下図 `3d_preview` の1エリアはトラッカーを表示するためのもので、トラッカーが選択されると、ここでトラッカーの方向をリアルタイムで表示できます。ボックスボタンをクリックすると、トラッカーのリアルタイムプレビューが閉じられます。

<a id="how_to_close_3d_preview"></a>

### 3Dを閉じてパフォーマンスを節約する方法
* 下図 `3d_preview` の2エリアのボタンをクリックすると、レンダリングを一時停止できます。レンダリングを一時停止することでリソースを節約できます。
* 3Dレンダリングを完全に閉じる方法：設定ページで3Dレンダリングを直接オフにでき、再起動後に有効になり、メモリを節約し、より大きな程度でCPUとグラフィックカードのリソースを節約できます。
