# 其它接入

### [雲鏡虛擬直播](https://fancyjing.com/home)

- 簡介
> 「雲鏡虛擬直播」是鄰境科技旗下的一款免費 3D 虛擬直播工具。只需一台手機或是帶有攝像頭的電腦，即可實現面部、半身及全身動捕，同時已經接入 `Rebocap` 動捕設備實現精確動作捕捉。內置豐富道具、舞蹈動作及玩法，為虛擬主播提供更便捷的直播服務。  
>   
> 注意：如果出現模型胸部和腰部拉長現象，請聯繫建模師修改骨骼，一般是由於 `Q` 版模型胸部骨骼沒有和 `mesh` 保持一致縮放導致

- 視頻教程
