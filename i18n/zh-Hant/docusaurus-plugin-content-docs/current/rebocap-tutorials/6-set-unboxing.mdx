## <span style="background: #f1e3ff; color: #000000; padding: 4px 12px; border-radius: 0 6px 6px 0; display: inline-block; margin-left: -20px; margin-bottom: -15px;">VR – 校準姿態圖鑑</span>

<!-- ==================== 步骤 1：A pose 开始 ==================== -->
<div style="display: flex; flex-direction: row; gap: 30px; flex-wrap: wrap; align-items: flex-start; margin: 25px 0;">
<div style="flex: 1; min-width: 250px; text-align: center;">


![A_pose-6]( ../../../../../static/img/unboxing/A_pose-6.jpg )


</div>
<div style="flex: 1.5; min-width: 300px" class="tutorial-step-text">
<strong style="font-size: 1.15em" class="tutorial-step-title">A姿態</strong><br />
- 抬起 VR 手柄，避免追蹤器捕捉到手柄中的磁鐵。<br />
- 保持適當的雙腳間距，不要併攏腳也不要過度張開，與圖中相近。<br />
- 自然放鬆的站立，不需要繃緊肌肉。
</div>
</div>
<!-- ==================== 步骤 1 结束 ==================== -->

<!-- ==================== 步骤 2：S pose 开始 ==================== -->
<div style="display: flex; flex-direction: row; gap: 30px; flex-wrap: wrap; align-items: flex-start; margin: 25px 0;">
<div style="flex: 1; min-width: 250px; text-align: center;">


![S_pose-6]( ../../../../../static/img/unboxing/S_pose-6.jpg )


</div>
<div style="flex: 1.5; min-width: 300px" class="tutorial-step-text">
<strong style="font-size: 1.15em" class="tutorial-step-title">S姿態</strong><br />
- 半蹲 + 彎腰並低頭，讓追蹤器透過前傾的角度獲知身體的前方朝向。<br />
- 儘量保持雙腿平衡與膝蓋間距。<br />
- 如果感覺彎腰困難、不明顯，可以使用進階校準。
</div>
</div>
<!-- ==================== 步骤 2 结束 ==================== -->

<!-- ==================== 步骤 3：B pose 开始 ==================== -->
<div style="display: flex; flex-direction: row; gap: 30px; flex-wrap: wrap; align-items: flex-start; margin: 25px 0;">
<div style="flex: 1; min-width: 250px; text-align: center;">


![B_pose-6]( ../../../../../static/img/unboxing/B_pose-6.jpg )


</div>
<div style="flex: 1.5; min-width: 300px" class="tutorial-step-text">
<strong style="font-size: 1.15em" class="tutorial-step-title">B姿態 (進階校準)</strong><br />
- 把 S Pose 的彎腰姿勢拆分為單獨的彎腰動作進行記錄。<br />
- 可以避免因 S 姿態引起的身體畸變<br />（常見於坐下後骨架系統向側面傾斜）。
</div>
</div>
<!-- ==================== 步骤 3 结束 ==================== -->

</div>
