# 硬體列表資訊
硬體列表資訊用於全域預覽。
- 第一列
  > 部位名稱
- 第二列
  > 部位編號，和感測器後面的標牌中的編號對應
- 第三列
  > 電池電量，電量百分比只能用於參考，這裡是通過電壓轉換而來，但是電壓測量不是專業 A/D 採樣測量，因此有一定誤差，不是絕對百分比，以具體使用時長為準
- 第四列
  > 這裡指的是通信品質，比如最近2秒中，120個幀率數據有多少百分比接收成功，**不是信號強度**！信號強度在追蹤器詳細資訊中查看，單位是 `dbm`，一般 >-70dbm 認為信號良好，比如-30 dbm 就是很強的信號
- 第五列
  > 磁場絕對值大小，單位是 uT，磁場一般一致性好就是顯示綠色，如果個別很高或者很低會顯示紅色，磁場怎麼看，具體請查閱 磁場檢測和診斷

# 硬體詳細資訊
點擊列表中某一項，即可查看硬體詳細資訊。

<a id="close_single_tracker"></a>

### 單獨關機
  > 可以點擊關機按鈕，單獨關閉這個追蹤器

<a id="detail_information"></a>

### 其它資訊
- 電池電量
  > 具體電量百分比，這裡僅僅用於大致參考，具體以使用時長為準
- 通信品質
  > 最近2秒中，120個幀率數據有多少百分比接收成功，不是信號強度！信號強度在追蹤器詳細資訊中查看，單位是 `dbm`，一般 >-70dbm 認為信號良好，比如-30 dbm 就是很強的信號
- 接收信號
  > 接收器收到的感測器的信號強度，>-70 dbm 認為是信號良好
- 被動信號
  > 感測器收到的接收器的信號強度，>-70 dbm 認為是信號良好
- 相對磁場
  > 這裡是相對於校準時 A-Pose 所在空間和時刻的相對大小，相對大小持續在 1.1 以內一般認為磁場環境良好，磁場環境良好的情況下，應用抗磁算法效果更好，具體磁場判定和磁場校準，請看這裡
- 加速度
  > 三軸校準後加速度，歸一化到當前重力加速度中
- 陀螺儀
  > 三軸校準後陀螺儀，如果追蹤器完全靜止，這裡的數值離0點超過0.2，建議進行 陀螺儀校準
