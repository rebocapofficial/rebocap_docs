# 3D 預覽功能
- **展示動捕即時預覽畫面**

  **操作說明**
  * 滑鼠左鍵移動視角
  * 滑鼠中鍵拖動相機位置，也就是拖動視角
  * 滑鼠滾輪是按照當前方向，向前向後移動相機位置，可以用於調節視圖中人物大小
- **展示追蹤器**
    > 下圖 `3d_preview` 中標註的 1 區域是用於展示追蹤器，當追蹤器被選中以後，這裡可以即時展示追蹤器方向，點擊盒子按鈕以後，則關閉追蹤器即時預覽

<a id="how_to_close_3d_preview"></a>

### 如何關閉3D節省性能
* 點擊下圖 `3d_preview` 中2區域按鈕，可以暫停渲染，暫停渲染能夠節省資源
* 徹底關閉3D 渲染方法：配置頁面中可以直接關閉3D渲染，重啟後生效，能夠節省記憶體，更大程度節省 CPU 和 顯卡 資源
