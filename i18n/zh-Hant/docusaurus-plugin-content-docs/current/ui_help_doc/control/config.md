### 1. 韌體更新
- 當前版本：最先點亮的追蹤器的標號版本，如果版本不一致，校準的時候，日誌中會顯示不同追蹤器版本信息，23年12月之前購買的如果沒有更新過韌體，這裡的版本號可能混亂。
- 可用版本：從服務端拉取可用版本號，防止個別版本發布後存在問題，方便用戶回滾，目前 `preview` 只有 `v4` 版本，右側刷新圖標點擊會重新拉取
- 更新韌體
    > 韌體更新到選擇的版本，這裡需要保證連接的追蹤器電量充足並且全部連接成功，左下角側全部100%信號（更新過程中信號質量會有波動屬於正常現象）。下邊是更新進度顯示，更新完成後，全部亮白燈。如果發生更新錯誤，或者更新後 RGB 燈不是白色，可以再次更新。更新以後，需要重啟追蹤器才能生效，按追蹤器的按鈕或者關閉追蹤器再打開，均可重啟。

:::info 韌體更新可以反覆更新，但不建議更新好的情況下，繼續更新

:::

### 3. 追蹤器校準

<a id="gyro_calibrate"></a>

#### 陀螺儀校準
陀螺儀校準相對簡單，將追蹤器點亮以後，靜止放置於地面即可校準，注意，一定要絕對靜止，不可有震動，例如遠離台式機機箱（機箱上有風扇引起的震動）

:::info 一般情況下陀螺儀不用反覆校準，如果溫差比較大，可能需要校準，使用6軸時，也建議校準。

:::


<a id="magnet_calibrate"></a>

#### 磁場校準
磁場校準相對繁瑣，耗時比較久。**但是磁場校準是必須了解的內容！！！！！**

:::info 為什麼需要磁場校準


電池壽命會隨著使用衰減，導致電池內部元素發生變化，並且充電過程中，極少數部分元件攜帶微量鐵元素，也可能存在磁化現象，因此 PCB 電路板自身的磁場可能發生變化，磁場校準主要用於消除傳感器自身所攜帶的磁場，也就是文檔所提到的磁偏。

:::


:::info 怎麼判斷是否有磁偏（即什麼時候需要磁場校準）


可以在空間某個固定點位6個面翻轉傳感器，觀察傳感器磁場值的大小，如果相差在2uT以內，一般磁偏較小，或者[相對磁場相差 0.1](../info#detail_information) 以內。

**選擇翻轉的點位必須遠離所有磁場來源，比如運動磁吸手環，手機，電腦，耳機、耳麥、帶磁金屬等**

:::


如何校準磁場：
  - 將追蹤器放到充電盒子中，**間隔方式，並且充電觸點向上**，充電觸點是指的追蹤器三個黃色的金屬點位（即放置方向和充電時相反）
    > 一次最多放置**8個**，一定要**間隔放置**，並且**充電觸點向上**，否則會將盒子攜帶的磁場校準到追蹤器中！！！
  - 校準的時候不要放在桌子上，**建議在空中固定點位校準**，即充電盒子盡量圍繞空間固定點位旋轉。
    > 提示，即使是木質的桌子，很多也是有鐵螺絲的或者鐵釘的。
  - 總共6個面，點擊磁場校準以後，一定要跟隨文字信息旋轉盒子，盡量勻速旋轉，總計6個面，每個面旋轉時間預留是6秒左右，切換旋轉面1秒左右，一定要在規定時間內完成，否則影響校準效果。
    > 比較極端情況，如果只在規定時間內旋轉了3個面，基本上校準後，整體效果越來越差，可以參考上邊說的 怎麼判斷是否有磁偏 來檢查校準結果。

### 4. 通信頻道修改
- 大部分用戶不需要用到，默認的通信通道足夠大部分人使用，這裡主要解決部分用戶在同一個空間內兩套設備同時使用的需求。
    > 對於有更高要求的，未來會推出商業版本軟件，主要針對企業以及多主播同台聯動使用，加上免費通信頻道，最高支持10台設備同一個空間使用。如有更高的定制需求，也可以直接發送郵件聯繫我們。

- 功能說明
  - 左側按鈕是重置頻道按鈕，也就是出廠的默認通信頻道，右側是可供選擇的免費頻道，不建議頻繁修改頻道。
  - 每次建議所有傳感器全部連接以後再修改頻道，否則會導致沒有修改頻道的跟踪器無法和接收器匹配
  - 如果匹配失敗，並且不清楚如何連接，可以使用充電底倉重置跟踪器頻道，接收器重置頻道可以使用左側重置頻道按鈕重置即可（如果接收器已經屬於默認頻道，左側按鈕將會不可點擊）

- 充電盒子重置頻道的方法
  > 將跟踪器放入充電盒子中，方向和充電時保持一致，然後單擊充電盒子按鈕點亮所有跟踪器，然後長按充電盒子按鈕8秒後鬆開，發現跟踪器藍燈閃爍，然後再長按8秒鬆開，綠燈閃爍意味著放在充電盒子裡的跟踪器重置通信頻道成功。

- 如果僅僅是接收器無法連接，[請查看這裡](../../tutorial/connect_and_use#how_to_solve_cannot_connect)

### 5. 更新追蹤器 RGB 顏色
更新追蹤器上 `RGB` 顏色，如果希望不同的追蹤器有不同的顏色，可以一個一個開啟後更新。左側圖標是顏色示例，但是由於 RGB 螢幕顏色和實際燈光顏色存在差異，以最終RGB燈光顏色為準。


:::warning 耗電提醒


RGB中數值越高，燈越亮，耗電越快，燈越多，耗電越快！！！比如白色的最亮的燈光耗電是最快的，續航最短。

:::


<a id="update_reject_mag_and_strenth"></a>

### 6. 更新追蹤器抗磁強度以及信號發射功率
- 抗磁強度一般建議調節為12不要動（數值範圍是 1~12），其它數值可能存在追蹤器抖動嚴重現象。
- 發射功率可調節範圍是 1~18，18對應9.1dbm，1對應 -8dbm。18的時候發射功率更大，一般調節到1，會影響通信距離，但是相應的續航能夠提升1~2小時。

# 系統配置面板
<img src="/img/config_software.png" alt="系統配置面板" />

### 1. 數據記錄診斷
用於反饋給 `rebocap` 開發者動作原始加密後的數據，方便開發者根據錄製的動作，診斷問題，提升動補質量。默認系統會自動在動作校準以後開始記錄數據，但是不會保存，用戶點擊開始記錄，然後再次點擊停止錄製，數據會自動保存到 `<rebocap_install_dir>/data/record_data/` 目錄下

### 2. 系統語言選擇
選擇語言，這裡會根據系統自動選擇當地語言，如果不支持，則可以默認使用英語。

註：目前其它語言是根據 中文 自動翻譯得到，如有疑問，可以在論壇中反饋改進。

### 3. 主題色彩
當前版本支持黑色主題和白色主題，其中白色主題目前尚未完善，後續會繼續改進。

### 4. 系統快捷鍵
方便用戶使用快捷鍵綁定，目前只支持vr下的操作，錄製功能未來增加。

點擊以後，鍵盤輸入快捷鍵即可綁定，注意，只能以以下四個功能鍵開頭：`win、ctrl、shift、alt`，並且最多支持兩個功能鍵組合，然後必須搭配一個普通字符，比如 `a~z`

<a id="websocket_broadcast"></a>

### 5. websocket 廣播
所有SDK、插件等，都是用 `rebocap` websocket廣播，如果希望修改端口號，可以關閉廣播以後修改，再開啟。

:::danger 端口被佔用


如果默認端口7690被佔用，那麼這裡的端口會自動+1，如果發現插件無法連接，可以查看這裡的端口號是否有變更！！！

:::


### 6. 3D 渲染開關
用於開啟或者關閉 3D 預覽窗口，關閉可以節省性能，但是不方便動補調試。
### 7. 使用系統標題欄開關
使用系統標題欄以後，整體風格樣式可能不統一，但是性能原則上會更好一點點。
### 8. 軟體更新
功能預留，未完成熱更新功能
