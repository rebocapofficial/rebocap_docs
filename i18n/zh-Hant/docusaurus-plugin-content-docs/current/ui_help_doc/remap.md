# 傳感器整體預覽

如下圖所示，對應關節部位綠色則代表傳感器已連接，灰色則表示未連接。圖中左側即為人體自身的左側，比如下圖中左腳的傳感器沒有連接，右邊下腿的傳感器也沒有連接。

## 傳感器替換功能
這裡可以透過傳感器點位拖拽，實現傳感器重組

:::warning 替換以後，如果沒有點亮，會導致校準失敗


很多用戶不小心拖拽以後，比如將左手拖拽到左上腿，但是校準的時候又沒有開啟左手，會提示校準失敗，因為左上腿沒有開啟！！！
拖拽以後，被替換的點位不管有沒有被點亮，都是不生效的。

:::


### 拖拽演示
> 在節點上按住滑鼠左鍵並保持，拖拽到目標點位即可。

### 能夠替換其它部位的點位
   * 左手   (13號節點)
   * 右手   (14號節點)
   * 左下臂 (11號節點)
   * 右下臂 (12號節點)


### 能夠被替換的部位
   全部


### 特別功能
   * 替換為肩膀位置
    > 這裡如果希望肩膀部位更加靈活，可以替換為肩膀位置，但是對綁帶要求比較高，需要自行解決。


### 替換示例
如下圖所示，左右手四個點位全部替換了，替換如下部位
* 左肩
* 右肩
* 左下腿
  > 左下腿替換以後沒有被點亮，因此會發生校準失敗的現象！！！
* 右下腿
