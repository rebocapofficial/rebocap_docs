## 必須了解項
- [磁場校準](../ui_help_doc/control/config#magnet_calibrate)

:::danger 首次充電一般需要做磁場校準，後續建議間隔一段時間完成一次磁場校準


- 磁場校準看可以參考文件中（上邊磁場校準鏈接下方），和社區合作完成的校準視頻。
- 磁場校準必須掌握，並且其中注意事項非常重要，如果未按照注意事項來，那麼校準結果將會錯誤，磁場校準可以反覆校準。

:::

## 視頻教程
影片教學為輔助選項，文件的介紹更為詳細，並且會提到過程中遇到的問題。另外，影片教學為基礎教程，不包含連結到第三方軟體，例如 [SteamVR 以及其他第三方直播軟體連接請查看這裡](https://doc.rebocap.com/zh_TW/tutorial/connect_and_use.html#third_party)，[插件如 Blender、Unity、UE 等請查看這裡](../plugins/plugins)，更多問題可以到[社區提問](../README#community)

:::danger 如果需要更好的效果，或者遇到各種問題，請完整閱讀文字版本教程


- 比如腳底穿戴方式影響很大
- 比如是否需要開啟抗磁
- 比如是否需要打開 AI 引擎
- 怎麼和其它軟體連接

:::
