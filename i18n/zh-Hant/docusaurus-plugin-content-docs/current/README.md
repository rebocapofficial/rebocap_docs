## <p>特別聲明</p>
<p>翻譯部分基於中文使用 `chatgpt` 翻譯，如有任何疑問，請在論壇或者社區反饋，我們會修正不恰當描述！</p>

> 注意：軟體必須在聯網狀態下使用，如果希望離線使用，請使用手機熱點網路，開啟軟體30秒以後再斷開網路即可（可以查看日誌，只要日誌顯示網路校驗成功，即可斷開網路）。
