# Other Access

### [Yunjing Virtual Live Streaming](https://fancyjing.com/home)

- Introduction
> “Yunjing Virtual Live Streaming” is a free 3D virtual live streaming tool developed by Linjing Technology. With just a mobile phone or a camera-equipped computer, you can achieve facial, half-body, and full-body motion capture, and it has already integrated the `Rebocap` motion capture device for precise motion tracking. A rich collection of props, dance animations, and gameplay is built in, providing VTubers with more convenient live streaming services.  
>
> Note: If you encounter stretching in the bust or waist of the model, please ask the modeler to adjust the skeleton. This is usually caused by the bust bone of a chibi (`Q`-style) model not being scaled consistently with the mesh.

- Integration documentation
