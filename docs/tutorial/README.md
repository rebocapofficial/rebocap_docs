## Must-Know Items
- [Magnetic Field Calibration](../ui_help_doc/control/config#magnet_calibrate)

:::danger Magnetic field calibration is generally required for the first charge, and it is recommended to perform magnetic field calibration at intervals thereafter


- For magnetic field calibration, you can refer to the document (below the magnetic field calibration link above) and the calibration video completed in cooperation with the community.
- Magnetic field calibration must be mastered, and the precautions are very important. If the precautions are not followed, the calibration results will be incorrect. Magnetic field calibration can be repeated.

:::
