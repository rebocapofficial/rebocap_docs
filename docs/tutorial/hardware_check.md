- About Hardware Quality
  1. There is a certain gap between the upper and lower covers of the tracker, and a slight misalignment between the covers, which are normal phenomena.
  2. In the large charging box, the injection molding water outlet of the middle support part is left on the appearance surface, resulting in a certain gap and defect next to the buckle.
  3. The USB interface of the receiver is slightly tilted from the factory, which does not affect any use.
  4. The plastic shell currently has no UV plating, making it easy to leave scratches.
  5. There is a certain capacity error between different tracker batteries, which is normal. The usage time is the standard [the battery percentage can only be used as a reference, not as an evaluation indicator].

## Charging and Usage Time Instructions

- When you first receive it, you need to confirm whether it can be charged. Normally, after plugging in the adapter, all trackers will light up red, and turn green when fully charged.
  > 
- If you find no lights on, check if the tracker has become loose during transportation, reinsert the corresponding tracker, and if none of them light up, check if the power adapter is powered.
- Normally, it takes about 2 hours to fully charge, and the usage time is about 30 hours.
:::info Usage Time and Charging Instructions


1. The usage time is related to the ambient temperature, and low temperatures will reduce the usage time.
2. If individual nodes do not shut down normally, the power will continue to be consumed, resulting in a significant difference in usage time compared to other trackers.
3. After fully charged, removing and reinserting the tracker will show a red light again, which is normal. It is similar to traditional mobile phones or other electrical appliances charging, where after being fully charged, removing and recharging can still perform short-term trickle charging.

:::

- <p>In rare cases, some trackers may not turn green after being fully charged. Please refer to the battery level displayed by the software after connection!</p>
- <p>When left unused for a long time, it is recommended to discharge and recharge every 2~3 months, otherwise, the battery life will be affected! For example, VR users may have 5~7 idle points, which can be replaced with the forearm if there are damaged points in the future, so it is recommended to maintain the battery properly.</p>

> Note: The usage time is not related to the human motion state. Once connected, regardless of the speed of movement, the current consumption remains consistent.

## Tracker Status Description
|       | Standby State | Disconnected State | Connected State | Low Battery | Firmware Update | Firmware Update Successful |
|:-----:|:-------------:|:------------------:|:---------------:|:-----------:|:---------------:|:--------------------------:|
| RGB Status | Breathing Light (3S) | Breathing Light (10S) | Fast Flashing | Red Light Flashing<br />Shuts down after 3 times | Blue Light Flashing | White Light Flashing |

- After the tracker lights up, it enters standby mode. In standby mode, the breathing light continues to flash for 30S, and it will automatically shut down if not connected for 10 minutes.
- As long as the receiver is powered, the tracker will enter the connected state, and it will not automatically shut down in the connected state.

:::warning Closing the software will not automatically shut down


In some cases, failure to shut down may also lead to continuous power consumption!

:::


- If the receiver is powered off, the tracker will switch back to standby mode.
- If the tracker does not automatically shut down, it will immediately enter the connected state once the receiver is powered.
