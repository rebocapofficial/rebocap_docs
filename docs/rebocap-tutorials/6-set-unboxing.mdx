# 4 - Install software - Check firmware update

## A - Install software

🌐Download link → [https://doc.rebocap.com/en\_US/tutorial/software\_install.html](https://doc.rebocap.com/en_US/tutorial/software_install.html)

- Currently recommended:\
  V02 Beta02 version, with default switches adjusted for VR and new algorithm maintaining orientation under strong interference sources.

- Recommended install on a non‑system drive (not C:).

## B - Connect to computer

- Insert USB receiver into computer,choose an open surrounding port.\
  Or use the included USB extension cable.\
  \
  If tracker signal does not stay at 100%, relocate the receiver.





- Click Connect in software（After beta02, it will auto connect）
