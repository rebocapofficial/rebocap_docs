## A - Install software

🌐Download link → [https://doc.rebocap.com/en\_US/tutorial/software\_install.html](https://doc.rebocap.com/en_US/tutorial/software_install.html)

- Version selection:\
  V01 - Magnetic field is stable, mainly for dancing.<br />
  V02 Beta02 - with default switches adjusted for VR and new algorithm maintaining orientation under strong interference sources.

<details>
<summary>If use V01 version, Some settings need to be change.</summary>

   &emsp;&emsp;1 - Turn off extra displayed tracker points.<br />
   &emsp;&emsp;&emsp; Open [Set 'SteamVR'Nodes] → Turn off [Left/Right Upper Arm] 

   &emsp;&emsp;2 - Turn off the feature that stuck running globally.<br />
   &emsp;&emsp;&emsp; → [Motion Params] → Turn off [Vertical IK & Horizontal IK]


</details>





- Recommended install on a non‑system drive (not C:).
