<!-- ==================== Check Firmware End ==================== -->

<!-- ==================== Dropdown Start ==================== -->
<details>
<summary>View the supported firmware versions for the corresponding software.</summary>
   &emsp;&emsp; Some firmware versions have major algorithm changes and are incompatible with older software versions. <br /> 

   &emsp;&emsp; When switching back to older software, you need to downgrade the firmware accordingly.<br /> 

   &emsp;&emsp;&emsp; release_v01 - ◼️tracker: V6 / V7 , 📡receiver: V6 / V7 <br /> 

   &emsp;&emsp;&emsp; release_v02 beta02 - ◼️tracker: V15 , 📡receiver: V6 / V7 <br /> 

   &emsp;&emsp;&emsp; (Unreleased) release_v02 beta02.1 - ◼️tracker: V16 , 📡receiver: V8 <br /> 

</details>
<!-- ==================== Dropdown End ==================== -->

- Open the log window to view the actual firmware version of each tracker <br /> 
  (the log window is located under "Connect and Power Off" in the software).

- Trackers are updated wirelessly 📶 — no USB data cable is required.<br /> 
  🚫 Do not update the tracker and receiver at the same time.<br /> 
  If the tracker update fails, consider turning off half of the trackers to stabilize the data upload. Trackers with the correct firmware version in the logs can remain turned off.

- If the update fails, restart the tracker and click update again.<br /> 
  &emsp;&emsp;🟩Green light – Fast flash: Tracker is working normally<br /> 
  &emsp;&emsp;🟩Green light – Slow flash: Tracker is waiting for receiver signal<br /> 
  &emsp;&emsp;🟦Blue light: Tracker is receiving firmware data<br /> 
  &emsp;&emsp;🟨Yellow light: Update failed (manually press the 🔘 button to restart, then update again)<br /> 
  &emsp;&emsp;⬜White light: Update successful (usually restarts automatically after 10s; if it fails to restart automatically, a manual restart is required)<br /> 

- When the 📡receiver update is completed, disconnect and re-insert the USB, then 🔄restart the software.

- If bending is difficult or unclear, you can use Advanced Calibration.

## <span class="tutorial-heading-flag" style="background: #f1e3ff;">VR – Calibration Posture Guide</span>

<!-- ==================== Step 1: A pose Start ==================== -->
<div style="display: flex; flex-direction: row; gap: 30px; flex-wrap: wrap; align-items: flex-start; margin: 25px 0;">
<div style="flex: 1; min-width: 250px; text-align: center;">

![A_pose-6]( ../../static/img/unboxing/A_pose-15-vr.jpg )

</div>
<div style="flex: 1.5; min-width: 300px" class="tutorial-step-text">
<strong style="font-size: 1.15em" class="tutorial-step-title">A Pose</strong><br />
- Raise the VR controllers to avoid the trackers capturing the magnets in the controllers.<br />
- Keep proper foot spacing, neither too close nor too wide, similar to the image.<br />
- Stand naturally and relaxed, no need to tense your muscles.
</div>
</div>
<!-- ==================== Step 1 End ==================== -->

<!-- ==================== Step 2: T pose Start ==================== -->
<div style="display: flex; flex-direction: row; gap: 30px; flex-wrap: wrap; align-items: flex-start; margin: 25px 0;">
<div style="flex: 1; min-width: 250px; text-align: center;">

![S_pose-6]( ../../static/img/unboxing/T_pose-15-vr.jpg )

</div>
<div style="flex: 1.5; min-width: 300px" class="tutorial-step-text">
<strong style="font-size: 1.15em" class="tutorial-step-title">T Pose</strong><br />
- Spread your arms horizontally.<br />
- Sometimes muscle memory differs from actually stretching the hands straight, so take a visual check.
</div>
</div>
<!-- ==================== Step 2 End ==================== -->

<!-- ==================== Step 3: S pose Start ==================== -->
<div style="display: flex; flex-direction: row; gap: 30px; flex-wrap: wrap; align-items: flex-start; margin: 25px 0;">
<div style="flex: 1; min-width: 250px; text-align: center;">

![S_pose-6]( ../../static/img/unboxing/S_pose-15-vr.jpg )

</div>
<div style="flex: 1.5; min-width: 300px" class="tutorial-step-text">
<strong style="font-size: 1.15em" class="tutorial-step-title">S Pose</strong><br />
- Half squat + bend waist + lower head, so that the trackers can determine the forward direction of the body through the tilt angle.<br />
- Keep legs balanced + knee spacing, with arms parallel facing forward.<br />
- If bending is difficult or unclear, you can use Advanced Calibration.
</div>
</div>
<!-- ==================== Step 3 End ==================== -->

<!-- ==================== Step 4: B pose Start ==================== -->
<div style="display: flex; flex-direction: row; gap: 30px; flex-wrap: wrap; align-items: flex-start; margin: 25px 0;">
<div style="flex: 1; min-width: 250px; text-align: center;">

![B_pose-6]( ../../static/img/unboxing/B_pose-15-vr.jpg )

</div>
<div style="flex: 1.5; min-width: 300px" class="tutorial-step-text">
<strong style="font-size: 1.15em" class="tutorial-step-title">B Pose (Advanced Calibration)</strong><br />
- Separate the S Pose bending + head lowering into a separate bowing movement for recording.<br />
- This avoids body distortion caused by S Pose<br />(commonly seen as the skeleton system tilting to the side after sitting down).
</div>
</div>
<!-- ==================== Step 4 End ==================== -->

</div>
